Avaliao

Up 
get /minhaavaliacao
Retorna a avaliação do funcionário. (minhaavaliacaoGet)
Este endpoint retorna a avaliação de um funcionário em um determinado ano.

Query parameters

id (required)
Query Parameter — Messenger ID do funcionário.
ano (required)
Query Parameter — Ano da avaliação.

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Avaliação encontrada!

404

Não foi possível achar a avaliação.

default

Erro inesperado. Error

get /bancohora

Este endpoint retorna o banco de horas do funcionário, atrasos, faltas, etc.

ano (required)

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Banco de horas encontrado!

404

Banco de horas não encontrado no sistema.

410

default

Erro inesperado. Error
